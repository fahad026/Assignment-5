Question-2: What is the difference between innerHTML, innerText, and textContent?
  1. 1. innerHTML: When you need to insert or retrieve HTML content.
  2. 2. innerText: When you need only the visible text from an element.
  3. 3. textContent: When you need to get all text content, including hidden text.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a type of event propagation in which an event originates from the target element and moves up to its parent and ancestors in the DOM tree. It follows a bottom-to-top approach.
